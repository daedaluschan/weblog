# My AI Journal

Welcome to **My AI Journal** — a personal blog of all the bits and bites throughout my AI self-learning journey.

This site is built for mixed personal and technical notes: experiments, reflections, implementation logs, reading notes, and small discoveries worth saving.

## Start reading

- Visit the [Blog](blog/index.md) for posts in reverse chronological order.
- Browse [Tags](tags.md) when you want a topic-focused trail.
- Check generated archive and category pages as the post collection grows.

## Publishing workflow

Posts live under `docs/blog/posts/` as Markdown files. Write a post, commit it, push to GitHub, and the GitHub Actions workflow builds the site and deploys it to GitHub Pages.
